# cc-provide

Provi's Claude Code plugins.

## Plugins

| Plugin | Description |
|--------|-------------|
| [cc-iching](./cc-iching/) | Daily I Ching divination with authentic 3-coin casting |
| cc-reflection | Meta-cognitive reflection framework *(coming soon)* |

## Installation

Add the marketplace:
```bash
/plugin marketplace add pro-vi/cc-provide
```

Install a plugin:
```bash
/plugin install cc-iching
```
